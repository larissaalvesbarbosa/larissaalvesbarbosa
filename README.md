> "A vida sem música seria um erro, e sem tecnologia, seria estática."

Bem-vindo(a) ao meu universo! Sou uma eterna apaixonada por melodias que encontrou no código uma nova forma de compor o futuro. Atualmente, transformo café em sintonia e lógica no curso de **Desenvolvimento de Sistemas** no **SENAI Celso Charuri**.

### 🎼 Sobre mim

- 🎹 **Music Lover:** Se eu não estiver codando, com certeza estou com os fones de ouvido. A música me ensinou que a prática leva à perfeição — e no código não é diferente.
- 🎓 **Estudante no SENAI:** Aprendendo a base para construir sistemas que, assim como uma boa canção, facilitem a vida das pessoas.
- ✨ **Sonhadora:** Meu objetivo é unir tecnologia e arte para criar soluções que toquem o coração dos usuários.
- 🌱 **Evolução:** Acredito que aprender uma nova linguagem de programação é como aprender um novo instrumento: exige paciência, ritmo e muita paixão.
